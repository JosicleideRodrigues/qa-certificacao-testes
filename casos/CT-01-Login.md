# CT-01 - Login com dados válidos

## Objetivo
Verificar se o usuário consegue realizar login com dados válidos.

## Pré-condição
Usuário já cadastrado no sistema.

## Passos
1 Acessar a página de login
2 Inserir e-mail válido
3 Inserir senha válida
4 Clicar em botão "Entrar"

# Resultado esperado
O usuário deve ser redirecionado para a página inicial do sistema.
