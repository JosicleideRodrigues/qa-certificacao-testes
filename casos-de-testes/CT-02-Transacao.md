# CT-02 - Realizar Transação
## Objetivo
Validar se o usuário consegue realizar uma transação com sucesso.
## Pré-condição
Usuário logado no sistema.
## Passos
1 Acessar menu "Transferência"
2 Informar conta destino válida
3 Informar valor válido
4 Confirmar transação

## Resultado esperado
Sistema deve exibir mensagem de sucesso e registrar a transação.
